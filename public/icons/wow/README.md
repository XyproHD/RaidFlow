# WoW TBC Icons

Icons für Klassen und Specs werden hier abgelegt.

## Klassen (`classes/`)

- Dateiname: `<classId>.png` (z. B. `7.png` für Schamane)
- Class-IDs siehe `lib/wow-tbc-classes.ts` (z. B. 1=Warrior, 2=Paladin, …, 9=Warlock)

## Specs (`specs/`)

- Dateinamen wie in `lib/role-spec-icons.ts` unter `SPEC_ICON_FILES` (z. B. `fire-mage.png`, `restoration-shaman.png`)

Icons aus `C:\tmp\wow` können hierher kopiert werden; die Unterordner-Sortierung und das Matching erfolgen über die Dateinamen.
